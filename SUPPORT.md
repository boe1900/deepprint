# Support

如果你在使用或开发 DeepPrint Studio 时遇到问题，可以按下面方式处理：

- Bug 报告：提交 GitHub Issue，并附上复现步骤、日志和环境信息
- 功能建议：提交 GitHub Issue，说明业务场景和预期收益
- 使用咨询：如果仓库已开启 GitHub Discussions，优先发起 Discussion；否则可以先提交 Issue
- 安全问题：不要公开提 Issue，请先阅读 [SECURITY.md](./SECURITY.md)

提问前建议先查看：

- [README.md](./README.md)
- [docs/development-and-operations.md](./docs/development-and-operations.md)
- [docs/system-design.md](./docs/system-design.md)
